# 1️⃣ 속도는 “업로드”가 결정해요 📤

많은 분이
다운로드 속도만 봐요

하지만 파일을 보내는 쪽은
업로드 속도가 핵심이에요

✔ 10GB 이하 → 20Mbps 이상 권장
✔ 30GB 이상 → 50Mbps 이상 권장

특히 **p2p**는
보내는 쪽과 받는 쪽
둘 다 중요해요

fileyo, p2p, 파일공유
대용량이라면
업로드 먼저 점검하세요

# 3️⃣ 유선 연결은 왜 빠를까요? 🔌

와이파이는
전파 간섭을 받아요

유선은
직접 연결이에요

✔ 지연 감소
✔ 패킷 손실 감소
✔ 속도 안정성 증가

30GB 이상이면
유선이 체감 차이 커요

특히 p2p 전송은
지연에 민감해요

---

# 4️⃣ 패킷 손실이 속도를 떨어뜨려요 📉

공유기와 멀어지면
신호는 보이지만

패킷 손실이 생겨요

이때 전송은
다시 보내기를 반복해요

그래서 속도가
급격히 느려져요

공유기 근처 이동만으로도
속도가 올라가요
